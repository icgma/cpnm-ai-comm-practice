# 信息提取课程图片素材说明（更新版）

## 概述
本目录包含为《第8周-结构化提取》教学幻灯片添加的图表和可视化素材，旨在帮助本科生更好地理解信息提取的核心概念和技术。

## 图片文件说明

### 1. information_extraction_pipeline.png
**信息提取流程图**
- 展示了从非结构化文本到结构化知识的完整转换过程
- 包含NER、RE、EE三个核心步骤的串联关系
- 适合用于课程介绍和总结环节

### 2. ner_example.png  
**命名实体识别示例**
- 用颜色编码标注了不同类型的实体（人名、地名、职位等）
- 直观展示NER如何从文本中识别和分类实体
- 适合讲解NER概念时使用

### 3. relation_extraction_example.png
**关系提取示例**
- 展示了实体间关系的识别过程
- 包含三元组结构的可视化表示
- 适合讲解关系提取原理时使用

### 4. event_extraction_example.png
**事件提取示例**
- 展示了事件触发词和事件要素的识别
- 包含事件框架的结构化表示
- 适合讲解事件提取概念时使用

### 5. structured_vs_unstructured_data.png
**结构化vs非结构化数据对比**
- 对比了两种数据类型的特点和差异
- 解释了为什么需要信息提取技术
- 适合课程开场和动机说明时使用

## 教学设计原则

### 🎯 内容分离原则
每个图表都有独立的展示页面，确保：
1. **避免信息过载**：图表与详细解释分离
2. **循序渐进**：先讲概念，再展示图表
3. **重点突出**：每个图表有专门的说明页面
4. **符合认知规律**：从抽象到具体的展示顺序

## 教学优势

### 📚 对本科生的学习价值
1. **降低认知负荷**：每个页面专注一个核心概念
2. **视觉化学习**：抽象概念通过图表变得直观
3. **记忆强化**：一致的颜色编码和视觉元素
4. **循序渐进**：从动机→概念→实战→总结的逻辑顺序

### 👨‍🏫 对教师的教学价值
1. **结构清晰**：独立图表页面便于教学节奏控制
2. **灵活使用**：可以根据时间安排选择展示内容
3. **重点突出**：每个图表都有专门的解释页面
4. **前后呼应**：开场和总结形成学习闭环

## 使用建议

### 课前准备
- 预览所有图表页面，熟悉展示顺序
- 准备每个图表的详细解释和课堂讨论问题

### 课堂教学
- 先讲解概念，再展示图表，避免一次性信息过载
- 鼓励学生观察图表，提出问题和见解
- 利用图表进行互动讨论和知识检查

### 课后复习
- 学生可以通过图表回顾核心概念
- 图表作为记忆锚点，帮助长期记忆形成

## 技术规格
- **分辨率**：300 DPI，适合投影和打印
- **格式**：PNG，支持透明背景
- **尺寸**：针对幻灯片显示优化
- **颜色**：遵循无障碍设计原则，对比度良好
