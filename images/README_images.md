# Information Extraction Presentation Images

This directory contains educational images for teaching information extraction concepts to undergraduate students. All images are appropriate for academic use and teaching purposes.

## Generated Educational Diagrams

### 1. `information_extraction_pipeline.png` (200KB)
**Purpose**: Shows the complete information extraction workflow
**Content**: 
- Input: Unstructured text
- Processing steps: NER → Relation Extraction
- Output: Structured knowledge
- Example: "Apple Inc. was founded by Steve Jobs in Cupertino"
**Educational Value**: Perfect for introducing the overall IE process

### 2. `ner_example.png` (155KB)
**Purpose**: Demonstrates Named Entity Recognition with concrete examples
**Content**:
- Sample sentence with highlighted entities
- Entity classification (PERSON, LOCATION, TITLE)
- Visual color-coding for different entity types
**Educational Value**: Helps students understand how NER identifies and categorizes entities

### 3. `structured_vs_unstructured_data.png` (305KB)
**Purpose**: Compares structured and unstructured data characteristics
**Content**:
- Side-by-side comparison table
- Examples of each data type
- Shows how IE bridges the gap
**Educational Value**: Essential for understanding why IE is needed

### 4. `relation_extraction_example.png` (193KB)
**Purpose**: Illustrates how relationships between entities are identified
**Content**:
- Example: "Microsoft acquired LinkedIn for $26.2 billion in 2016"
- Shows entity relationships and additional information extraction
- Common relation types (works_at, founder_of, acquired, etc.)
**Educational Value**: Demonstrates relationship identification beyond just entities

### 5. `event_extraction_example.png` (201KB)
**Purpose**: Shows event extraction with trigger words and arguments
**Content**:
- Example: "Apple will launch the new iPhone 15 at a special event in Cupertino next month"
- Event components: trigger, agent, product, location, time
- Common event types (Business, Politics, Sports, Technology, etc.)
**Educational Value**: Introduces event extraction as a specialized IE task

## Usage Recommendations

### For Introductory Lectures:
1. Start with `structured_vs_unstructured_data.png` to establish the problem
2. Use `information_extraction_pipeline.png` to show the solution approach
3. Follow with specific technique diagrams (NER, relation extraction, event extraction)

### For Hands-on Exercises:
- Use `ner_example.png` for entity recognition practice
- Use `relation_extraction_example.png` for relationship identification exercises
- Use `event_extraction_example.png` for event analysis activities

### For Assignment/Project Inspiration:
- Reference `knowledge_graph_example.png` for final output visualization
- Use `data_transformation_before_after.png` to show transformation goals

## Technical Notes

- All generated diagrams are high-resolution (300 DPI) PNG files
- Color-coded for accessibility and visual learning
- Include both Chinese and English text where appropriate
- Designed for projection in classroom settings
- File sizes optimized for web and presentation use

## Creation Method

The custom diagrams were created using Python's matplotlib library with the following approach:
- Educational color schemes (light backgrounds, clear contrasts)
- Step-by-step visual progression
- Real-world examples relevant to students
- Consistent visual language across all diagrams

These images provide a comprehensive visual foundation for teaching information extraction concepts to undergraduate students, covering both theoretical foundations and practical applications.
