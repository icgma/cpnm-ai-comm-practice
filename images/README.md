# 信息提取课程图片素材说明

## 概述
本目录包含为《第8周-结构化提取》教学幻灯片添加的图表和可视化素材，旨在帮助本科生更好地理解信息提取的核心概念和技术。

## 图片文件说明

### 1. information_extraction_pipeline.png
**信息提取流程图**
- 展示了从非结构化文本到结构化知识的完整转换过程
- 包含NER、RE、EE三个核心步骤的串联关系
- 适合用于课程介绍和总结环节

### 2. ner_example.png  
**命名实体识别示例**
- 用颜色编码标注了不同类型的实体（人名、地名、职位等）
- 直观展示NER如何从文本中识别和分类实体
- 适合讲解NER概念时使用

### 3. relation_extraction_example.png
**关系提取示例**
- 展示了实体间关系的识别过程
- 包含三元组结构的可视化表示
- 适合讲解关系提取原理时使用

### 4. event_extraction_example.png
**事件提取示例**
- 展示了事件触发词和事件要素的识别
- 包含事件框架的结构化表示
- 适合讲解事件提取概念时使用

### 5. structured_vs_unstructured_data.png
**结构化vs非结构化数据对比**
- 对比了两种数据类型的特点和差异
- 解释了为什么需要信息提取技术
- 适合课程开场和动机说明时使用

## 教学使用建议

### 课程开场（幻灯片3-4）
- 使用 `structured_vs_unstructured_data.png` 说明信息提取的必要性
- 使用 `information_extraction_pipeline.png` 展示整体流程

### 核心概念讲解（幻灯片6-8）
- 使用 `ner_example.png` 讲解命名实体识别
- 使用 `relation_extraction_example.png` 讲解关系提取
- 使用 `event_extraction_example.png` 讲解事件提取

### 实战环节（幻灯片14-16）
- 在每个任务level中重复使用对应的图表
- 帮助学生建立任务与图表的对应关系

### 课程总结（幻灯片36）
- 再次使用 `information_extraction_pipeline.png` 回顾完整流程

## 技术规格
- 分辨率：300 DPI，适合投影和打印
- 格式：PNG，支持透明背景
- 尺寸：针对幻灯片显示优化
- 颜色：遵循无障碍设计原则，对比度良好

## 教育价值
这些图表通过以下方式增强教学效果：
1. **视觉化抽象概念**：将抽象的NLP概念转化为直观的图形
2. **颜色编码学习**：使用一致的颜色方案帮助记忆
3. **分步骤展示**：将复杂过程分解为简单步骤
4. **实例驱动**：通过具体例子说明抽象原理

## 更新建议
如需更新或添加新的图表，建议：
1. 保持风格一致性（颜色、字体、布局）
2. 确保高对比度，便于投影显示
3. 添加简洁的标注和说明
4. 优化文件大小，确保快速加载
